# dev + REDPEN + PC 대문 확인

PC worktree(`welcome-pc`) 대문만 본다. `welcome-mo` / `src/welcome/mobile/**` 는 읽지도 수정하지도 않는다.

## 1. dev 서버

- 워크스페이스: `pdf-publish-proofread` (PC worktree)
- 이미 `5173`에서 Vite가 떠 있으면 **다시 띄우지 않는다**
- 안 떠 있으면: `npm run dev` (포트 5173, `http://127.0.0.1:5173/REDPEN/`)

## 2. 브라우저

- `cursor-ide-browser` MCP 사용
- 뷰포트 **1400×900** 이상으로 맞춘 뒤 이동:
  - `http://127.0.0.1:5173/REDPEN/`
- `browser_snapshot` + 필요 시 `browser_take_screenshot`으로 화면 확인
