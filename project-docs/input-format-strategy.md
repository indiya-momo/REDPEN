# 입력 형식 전략 — PDF · 한글 PDF · HWPX

**마지막 갱신:** 2026-06-05  
**상태:** 방향 문서. 베타 중 코드 대규모 변경 없음.

---

## 1. 왜 필요한가

| 세그먼트 | 주된 원고 형식 |
|----------|----------------|
| 출판 편집 (지금 베타) | 인디자인 등 **조판 PDF** |
| 웹소설·시나리오 (장기) | **한글 → PDF**, HWP/HWPX |
| 번역 (장기) | PDF, Word, TXT 혼재 |

확장 시장으로 가려면 **「텍스트 PDF만 잘 됨」** 만으로는 부족할 수 있다.  
다만 **베타 1달·유료화·마이페이지**가 먼저이고, 입력 레일 확장은 **데이터 보고 결정**.

---

## 2. 원칙 (모든 형식 공통)

- 원고 **본문은 서버에 저장하지 않음** ([`strategy-decisions-2026-06.md`](strategy-decisions-2026-06.md) §5).
- 클라우드에는 **프로젝트(기준 묶음)** 만.
- 검수 핵심은 **탐지 + 하이라이트** (자동 수정·PDF 재생성 X).

---

## 3. 선택지 비교

### A. PDF 강화 (현재 레일)

**대상:** 모든 PDF, 특히 **한글(한컴)에서 저장한 PDF**.

| 장점 | 단점 |
|------|------|
| PDF.js·하이라이트 **기존 엔진 그대로** | PDF마다 추출 품질 편차 |
| 베타·유료 출시와 **충돌 없음** | 사용자가 매번 PDF로보내기 번거로움 |

**이미 있는 것:** Hancom 공백 bbox 보정, `pdfPublishGate` 한글 추출 점검, cMap·재저장 안내.

**베타~유료 할 일:**

- [ ] 한글 2020/2022 등에서 뽑은 **실패·성공 PDF 샘플** 3~5종 수집  
- [ ] 업로드 직후 **추출 품질 경고** 눈에 띄게  
- [ ] 베타 설문: 원고 형식 (PDF / 한글 / 기타)

---

### B. HWPX 읽기 (Post-beta POC)

**대상:** `.hwpx` (ZIP + XML). 한글 원고 **직접 열기**.

| 장점 | 단점 |
|------|------|
| 원고 형식 그대로, PDF 재저장 생략 | **페이지·좌표·하이라이트** 재설계 필요 |
| 브라우저 로컬 파싱 가능 (본문 서버 X) | 한글 버전·서식·글상자 복잡도 |

**권장 POC 범위 (2주 가정):**

1. hwpx → **평문 추출** (브라우저)  
2. 기존 **일관성·맞춤법 검수** (목록 중심)  
3. 하이라이트 **없음 또는 2차**  
4. 확정 클러스터 → **프로젝트 규칙으로 저장**

**전체 스택 일괄 개발 금지.**

---

### C. TXT / UTF-8 (번역 측)

번역 원고가 TXT 비중이 크면 HWPX보다 **먼저** 검토할 수 있음.  
하이라이트는 PDF와 동일 이슈 → 1차는 **목록 검수**.

---

## 4. 결정 트리 (웹소설·번역 타진 전)

```text
베타 로그 + 설문
    │
    ├─ 「한글 파일 열어달라」다수 + 한글 PDF 불만 높음
    │       → HWPX POC (B) 또는 TXT (C) 중 비중 큰 쪽
    │
    ├─ 한글 PDF만 보완하면 충분
    │       → A 강화 (Hancom 테스트 스위트)
    │
    └─ 출판 PDF만 안정
            → A 유지, 확장 타진 연기
```

---

## 5. 일정과 product-spine 연동

| 시기 | 입력 전략 |
|------|-----------|
| **지금 ~ 베타 1달** | A only |
| **유료 7,900 / 10,900** | A + 한글 PDF 가이드 문서화 |
| **마이페이지·프로젝트 시연 가능** | 타진 시작, 동시에 **결정 트리** 실행 |
| **POC 승인 시** | B 또는 C **하나만** 파일럿 |

---

## 6. 비목표 (입력 레일)

- 서버에 원고 본문 저장  
- HWP 구버전(.hwp) 전면 지원 (hwpx 우선 검토)  
- OCR·스캔 PDF  
- HWPX → PDF 자동 변환 파이프 (초기)
